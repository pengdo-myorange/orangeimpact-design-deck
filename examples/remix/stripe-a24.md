# Stripe × A24

## Fintech pitch, with personality

Numbers backed by narrative. Numbers told like a film.

# Three acts

1. **Setup** — the financial system most people inherit is dull and adversarial.
2. **Conflict** — every API today still smells of compliance, not commerce.
3. **Resolution** — payments shouldn't sound like a bank. They should sound like the brand using them.
