# Glass Futurism
## Layered translucency, system-blue calm

Apple · Arc · Spotify — soft surfaces, subtle dividers, SF Pro stack.

---

# What changed

- Cards use `backdrop-filter: blur(30px) saturate(180%)` on rgba(255,255,255,0.72)
- Dividers only (0 1px 0 rgba(0,0,0,0.04)) — no heavy shadows
- SF Pro Text / Display stack with -0.02em tracking
- System-blue #007aff accent — one tint per surface

---

<!-- layout: bento -->

# Three anchors

:::card
### Translucency
#### Not transparency
Let surfaces layer over content beneath. Frosted, not see-through.
:::

:::card
### Hierarchy by tint
#### Not by scale
Color depth replaces size contrast where possible.
:::

:::card
### Touch-first type
#### 17px body
Body copy favors readability at arm's length — not display size.
:::

---

# Where this works

- Hardware-adjacent product stories (watch, headphones, vision)
- Spatial/immersive workflows where translucency implies depth
- Audio and ambient-computing UI keynotes
